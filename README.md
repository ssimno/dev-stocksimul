# 주식 모의 프로그램 개발
<br/>

**프로젝트 목표**

* 전략 시뮬레이션 종류에 게임 개발을 해보고 싶어서 간단한 주식 시뮬레이션 개발을 진행
1. 확률적 알고리즘 숙련
2. MVVM 패턴 활용

**개발 환경**

| 내용 | 버전 |
| :--: | :--: |
| 소프트웨어 프레임워크 | WPF |
| Microsoft Visual Studio Community 2022 | v17.4.5 |
| .Net Framework | v4.7.2 |
